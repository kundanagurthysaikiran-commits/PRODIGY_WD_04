# 💼 PRODIGY_WD_04 - Personal Portfolio Website

## 📌 Task Overview

This repository contains **Task 4** of my **Prodigy InfoTech Web Development Internship**.

The objective of this task is to design and develop a **responsive Personal Portfolio Website** that showcases personal information, skills, projects, education, and contact details using HTML, CSS, and JavaScript.

---

## 🚀 Features

- Responsive design for desktop, tablet, and mobile devices
- Attractive landing/home section
- About Me section
- Skills section
- Projects showcase
- Education details
- Contact section
- Sticky navigation bar
- Smooth scrolling navigation
- Interactive animations and hover effects
- Social media links

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
